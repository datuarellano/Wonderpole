# Pin Assignments

## LED power indicator
13 = ledPin

## Capacitive switch
30 = touchSwitchPin

## Voltage divider pitch controller
31/A12 = wiperPin

## Linear Potentiometer
32/A13 = volumePin

## Potentiometers
33/A14 = potPin1
34/A15 = potPin2
35/A16 = potPin3
36/A17 = potPin4

## Accelerometer
37 = SCL1
38 = SDA1
